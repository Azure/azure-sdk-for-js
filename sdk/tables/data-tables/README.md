# Tables client library for JavaScript

This package contains an isomorphic SDK (runs both in Node.js and in browsers) for Tables client.



Key links:

- [Source code](https://github.com/Azure/azure-sdk-for-js/tree/main/sdk/tables/data-tables)
- [Package (NPM)](https://www.npmjs.com/package/AzureTables)

## Getting started

### Currently supported environments

- [LTS versions of Node.js](https://github.com/nodejs/release#release-schedule)
- Latest versions of Safari, Chrome, Edge and Firefox.

See our [support policy](https://github.com/Azure/azure-sdk-for-js/blob/main/SUPPORT.md) for more details.


### Install the `AzureTables` package

Install the Tables client library for JavaScript with `npm`:

```bash
npm install AzureTables
```



### JavaScript Bundle
To use this client library in the browser, first you need to use a bundler. For details on how to do this, please refer to our [bundling documentation](https://aka.ms/AzureSDKBundling).

## Key concepts

### TablesClient

`TablesClient` is the primary interface for developers using the Tables client library. Explore the methods on this client object to understand the different features of the Tables service that you can access.
