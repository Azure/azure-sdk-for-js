# Release History
    
## 4.0.0-beta.5 (2026-06-24)
### Features Added

The @azure/arm-commerce package has been upgraded to a new SDK generation that provides layered APIs, smaller bundles, and improved ergonomics. Starting from version 4.0.0-beta.5, this release includes breaking changes.

To migrate existing applications, see the https://aka.ms/azsdk/js/sdk/migration. For more information, refer to the https://aka.ms/azsdk/js/sdk/quickstart.
