# Release History

## 1.1.0-beta.1 (2025-12-04)

### Features Added
- Added optional `cloudSetting` on `HealthDataAIServicesClient` to automatically target Azure sovereign clouds without custom endpoints.
- Exported new `AzureClouds` enum and `AzureSupportedClouds` string literal type for cross-cloud configuration scenarios.
- Generated additional Azure Resource Manager common models (operation metadata, error details, managed identities, private link resources) to align with the 2024-09-20 API version.

### Features Added
