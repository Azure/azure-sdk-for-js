# Azure Web PubSub Protocol Layer

> see https://aka.ms/autorest

## Configuration

```yaml
package-name: "@azure/web-pubsub"
title: GeneratedClient
description: Web PubSub Client
generate-metadata: false
license-header: MICROSOFT_MIT_NO_VERSION
output-folder: ../
source-code-folder-path: ./src/generated
input-file: https://raw.githubusercontent.com/Azure/azure-rest-api-specs/main/specification/webpubsub/data-plane/WebPubSub/stable/2022-11-01/webpubsub.json
add-credentials: false
package-version: 1.1.2
v3: true
hide-clients: true
use-core-v2: true
```
